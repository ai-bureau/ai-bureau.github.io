## База `Тезисы`

Хранит позиции AI Bureau и состояние поиска подтверждающих материалов.

| Поле | Тип |
|---|---|
| `Тезис` | Title |
| `Блок` | Select |
| `Статус пошуку` | Select |
| `Источники` | Relation на базу `Источники` |

## База `Источники`

Это прежняя база `Статьи`. Она хранит найденные Codex материалы и напрямую на
сайт больше не публикуется. Старые публикационные поля сохранены ради истории.

Ключевые поля:

| Поле | Назначение |
|---|---|
| `Заголовок`, `URL` | Оригинальный материал |
| `Мова` | Язык оригинального материала |
| `Саммари` | Честное содержание и связь с тезисом |
| `Тезиси` | Связанные позиции AI Bureau |
| `Додав`, `Дата додавання` | Происхождение записи |

## База `Публикации`

Одна карточка представляет одну полноценную статью сразу на UA и EN.

| Поле | Тип | Требование |
|---|---|---|
| `Назва` | Title | Обязательно; внутреннее имя |
| `Статус` | Select | Обязательно |
| `Заголовок UA` | Rich text | Обязательно |
| `Заголовок EN` | Rich text | Обязательно |
| `Опис UA` | Rich text | Обязательно |
| `Опис EN` | Rich text | Обязательно |
| `Slug` | Rich text | Обязательно; `a-z`, `0-9`, дефисы |
| `Дата публікації` | Date | Обязательно |
| `Опубліковано` | Date | Заполняет публикатор |
| `Тезиси` | Relation | Связанные тезисы |
| `Источники` | Relation | Использованные источники |

Варианты `Статус`:

- `Черновик`
- `На review`
- `Публікувати`
- `Опубліковано`

Полный текст хранится внутри страницы:

```markdown
# UA
Готовый украинский текст

# EN
Final English text
```

Поддерживаемые блоки: абзацы, H2, H3, списки, цитаты, ссылки, divider и code.
Другой блок останавливает публикацию, чтобы содержимое не потерялось.

## Запрос и обновление

Публикатор выбирает только:

```text
Статус = Публікувати
Опубліковано = пусто
```

После одного успешного GitHub-коммита с обеими версиями:

```text
Статус = Опубліковано
Опубліковано = текущее время
```

## Авторизации

Notion MCP используется агентами. Public API token из `actual.env` используется
публикатором и должен иметь доступ к `Публикации` и связанным страницам
`Тезисы`.

Переименование поля, служебного разделителя или статуса требует синхронного
изменения кода, тестов и документации.
