# Changelog

Значимые изменения проекта документируются здесь. Даты указаны в формате
`YYYY-MM-DD`.

## 2026-06-13

- Создан публичный репозиторий `ai-bureau/ai-bureau.github.io`.
- Сайт опубликован на <https://ai-bureau.github.io/>.
- GitHub Pages переключён на GitHub Actions workflow.
- Подключён и проверен Notion Public API.
- Настроен `actual.env`, исключённый из Git.
- Проведён полный интеграционный тест:
  `Notion → publisher → GitHub → Actions → Pages → Notion`.
- Проверена отложенная публикация; тестовый материал удалён после проверки.
- Добавлен подробный комплект проектной, эксплуатационной и агентской
  документации.
